# DNS設定チェックリスト - jigsawpuzzlesimple.com

このチェックリストを使って、DNS設定を順番に確認してください。

## ✅ チェックリスト

### 1. ドメインプロバイダー側の設定

- [ ] ドメインプロバイダーの管理画面にログインできた
- [ ] DNS設定画面を開いた
- [ ] **Aレコード1**: `@` → `185.199.108.153` を追加した
- [ ] **Aレコード2**: `@` → `185.199.109.153` を追加した
- [ ] **Aレコード3**: `@` → `185.199.110.153` を追加した
- [ ] **Aレコード4**: `@` → `185.199.111.153` を追加した
- [ ] DNS設定を保存した
- [ ] 設定変更が完了したことを確認した

### 2. GitHub Pages側の設定

- [ ] GitHubリポジトリ `https://github.com/yuharih/simplejigsawpuzzle` にアクセスした
- [ ] **Settings** > **Pages** を開いた
- [ ] **Custom domain** に `jigsawpuzzlesimple.com` が入力されている
- [ ] **Save** をクリックした

### 3. DNS設定の反映確認（30分〜1時間後）

以下のコマンドを実行して、DNSレコードが表示されるか確認してください：

```bash
dig jigsawpuzzlesimple.com +short
```

- [ ] コマンド実行結果に `185.199.108.153` などのIPアドレスが表示された
- [ ] または、`nslookup jigsawpuzzlesimple.com` でIPアドレスが表示された

### 4. GitHub Pagesの検証確認（数時間後）

- [ ] GitHubの **Settings** > **Pages** で「DNS check successful」と表示された
- [ ] 「Certificate is valid」と表示された
- [ ] **Enforce HTTPS** にチェックを入れた

### 5. 最終確認

- [ ] `https://jigsawpuzzlesimple.com` にアクセスできた
- [ ] サイトが正常に表示された
- [ ] HTTPS（緑の鍵マーク）が有効になっている

## 🔍 現在のエラー状況

**エラーメッセージ**: `InvalidDNSError` - Domain's DNS record could not be retrieved

**原因**: DNSレコードがまだ設定されていないか、設定が反映されていません

**解決方法**: 上記のチェックリストの「1. ドメインプロバイダー側の設定」から順番に実行してください

## 📝 設定値のまとめ

### Aレコード（ルートドメイン用）

```
ホスト名: @ または空白
値: 185.199.108.153
TTL: 3600
```

上記を4つのIPアドレスで繰り返し：
- `185.199.108.153`
- `185.199.109.153`
- `185.199.110.153`
- `185.199.111.153`

### CNAMEレコード（wwwサブドメイン用・オプション）

```
ホスト名: www
値: yuharih.github.io
TTL: 3600
```

## ⏰ タイムライン

1. **DNS設定完了後**: 30分〜1時間でDNSレコードが反映される
2. **GitHub Pages検証**: 数時間で「DNS check successful」と表示される
3. **SSL証明書発行**: 数時間で「Certificate is valid」と表示される
4. **サイトアクセス可能**: 上記が完了すると `https://jigsawpuzzlesimple.com` にアクセス可能

## 🆘 まだエラーが続く場合

1. **DNS設定を再確認**
   - 4つのAレコードがすべて追加されているか
   - IPアドレスが正確か（コピペミスがないか）

2. **時間を置いて再確認**
   - DNS設定の反映には時間がかかります
   - 最大48時間かかることがあります

3. **プロバイダーに問い合わせ**
   - DNS設定が正しく保存されているか確認
   - ネームサーバーの設定が正しいか確認

4. **詳細なDNS設定ガイドを参照**
   - `DNS_SETUP_GUIDE.md` を確認してください
