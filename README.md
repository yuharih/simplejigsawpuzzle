# シンプルジグソーパズル

シンプルで楽しいジグソーパズルゲームアプリです。

## セットアップ

### 必要な環境
- Node.js 18以上
- npm または yarn

### インストール

```bash
npm install
```

### サンプル画像の配置

**重要**: `public/images/` フォルダに以下の3つの画像ファイルを配置してください：
- `puzzle1.jpg` - 画像1
- `puzzle2.jpg` - 画像2  
- `puzzle3.jpg` - 画像3

**画像サイズの推奨**:
- **推奨サイズ**: **600×600ピクセル（正方形）**
- **最小サイズ**: 600×600ピクセル以上
- **最大サイズ**: 1200×1200ピクセル（高解像度ディスプレイ対応）
- **アスペクト比**: 1:1（正方形）を推奨（他の比率でも動作しますが、トリミングされます）
- **ファイル形式**: JPG、PNG、WebP対応

画像がない場合、画像選択画面に「画像を読み込めませんでした」と表示されます。

### 今日の画像の配置

`public/images/today/` フォルダに以下の3つの画像ファイルを配置してください：
- `today1.jpg` - 今日の画像1
- `today2.jpg` - 今日の画像2
- `today3.jpg` - 今日の画像3

**画像サイズ**: 上記と同じ推奨サイズ（600×600ピクセル以上、正方形推奨）

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

### ビルド

```bash
npm run build
```

## 機能

- **画像選択**: 3種類のローカル画像から選択
- 2つの難易度（25ピース / 49ピース）
- ジグソーパズル形状のピース（突起と凹み）
- ドラッグ&ドロップでピースを配置
- 正しい位置に配置されたピースは自動的に確定
- 完成時にお祝い演出を表示

## 技術スタック

- React 18
- TypeScript
- Vite
